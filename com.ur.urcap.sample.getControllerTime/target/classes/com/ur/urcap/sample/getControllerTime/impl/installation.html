Controller Time Server

 
Run Time Server
 
Add functions to Expression Editor.
 

The XML-RPC handle is "timeserver"

Example year = timeserver.getYear()

 

Following functions are implemented:

getYear() - Returns the current year. E.g. 2017

getMonth() - Returns the current month. E.g. 5

getDate() - Returns the current date in the month. E.g. 31

getHour() - Returns the current hour. E.g. 16

getMinute() - Returns the current minute. E.g. 58

getSecond() - Returns the current second. E.g. 32

getMillisecond() - Returns the current millisecond as floating point. E.g. 390.231

All times are equivalent to the time of the controller
